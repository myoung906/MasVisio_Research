# 시각재활 연구결과 웹페이지 개발 프로젝트 계획서

## 프로젝트 개요
- **프로젝트명**: 시각재활 연구결과 공유 웹사이트
- **목적**: 시각재활 연구성과를 바이어 및 스폰서에게 효과적으로 전달
- **작업 시작일**: 2025-08-04
- **현재 상태**: 프로젝트 계획 수립 중

## 대상 고객
### 바이어 (Buyers)
- 의료기기 제조업체
- 안과 클리닉 및 병원
- 시각재활 센터
- 의료기술 유통업체

### 스폰서 (Sponsors)
- 연구기관 및 대학
- 정부 연구지원기관
- 민간 투자자
- 의료기술 벤처캐피털

## 웹사이트 구조 계획

### 필수 섹션
1. **홈페이지 (index.html)**
   - 연구 개요 및 핵심 메시지
   - 주요 성과 하이라이트
   - CTA (Call to Action) 버튼

2. **연구 개요 (research-overview.html)**
   - 시각재활 연구 배경 및 목적
   - 핵심 연구 질문과 가설
   - 연구팀 소개

3. **실험 결과 (experimental-results.html)**
   - 주요 실험 데이터 및 통계
   - 시각적 데이터 표현 (차트, 그래프)
   - 임상적 의미와 효과성 증명

4. **기술적 사양 (technical-specs.html)**
   - 연구에 사용된 기술 및 장비
   - 프로토콜 및 방법론
   - 재현 가능성 정보

5. **상업적 가치 (commercial-value.html)**
   - 시장 잠재력 분석
   - 경쟁 우위 요소
   - ROI 예측 및 비즈니스 모델

6. **파트너십 기회 (partnership.html)**
   - 바이어를 위한 라이선싱 옵션
   - 스폰서를 위한 투자 기회
   - 협력 연구 제안

7. **연락처 (contact.html)**
   - 연구팀 연락처
   - 문의 양식
   - 미팅 스케줄링

## 기술 스택
### 프론트엔드
- HTML5, CSS3, JavaScript (ES6+)
- 반응형 웹 디자인
- 접근성 최적화 (WCAG 2.1 AA 준수)

### 백엔드 (PHP 활용)
- 연락처 폼 처리 (contact_handler.php)
- 간단한 데이터 관리

### 콘텐츠
- Markdown 기반 콘텐츠 작성
- 이미지 최적화 및 압축
- SEO 최적화

## 파일 구조 (다국어 지원)
```
mvr_webpage/
├── index.html                 # 언어 감지 및 리디렉션 페이지
├── robots.txt                 # 검색엔진 크롤링 설정
├── sitemap.xml               # SEO용 사이트맵
├── contact_handler.php       # PHP 폼 처리기
├── ko/                       # 한국어 버전
│   ├── index.html           # 한국어 홈페이지
│   ├── contact.html         # 한국어 연락처 페이지
│   └── [future pages]       # 추가 한국어 페이지들
├── en/                       # 영어 버전 (투자자 대상)
│   ├── index.html           # 영어 홈페이지 (투자 어필 강화)
│   └── [future pages]       # 추가 영어 페이지들
├── css/
│   └── style.css            # 통합 스타일시트 (다국어 지원)
├── js/
│   └── main.js             # 메인 JavaScript (언어 감지 포함)
└── assets/
    └── images/             # 이미지 파일
```

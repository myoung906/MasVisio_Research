# MasVisio Research 웹사이트

## 개요

시각재활 연구 성과를 바이어·스폰서에게 효과적으로 전달하기 위한 전문 B2B 웹사이트입니다.

## 핵심 목표

1. **연구 성과 공개**: 혁신적인 연구 결과의 국제적 공유
2. **비즈니스 기회 창출**: 바이어·스폰서와의 협력 기회 제시
3. **투자 유치**: 상업적 가치와 ROI를 통한 자금 조달

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js (데이터 시각화)

- Three.js (3D 렌더링, 선택적)

### 배포
- **호스팅**: GitHub Pages
- **CDN**: 글로벌 콘텐츠 전송
- **버전 관리**: Git

## 콘텐츠 섹션

1. **홈페이지**: 프로젝트 개요 및 주요 성과
2. **연구 분야**: 4D 근시, AI 진단, 맞춤형 솔루션
3. **성과 지표**: 임상 성공률, 시장 잠재력
4. **대상 고객**: 바이어·스폰서 정보
5. **연락처**: 파트너십 문의 폼
